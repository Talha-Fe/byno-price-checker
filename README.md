# Önemli

Bu proje %80 oranında AI kullanılarak yapılmıştır. <br>
Olası hatalar ortaya çıkabilir.  <br>
Normal durumlarda DOTA2, CS2, RUST ve TF2 için çalışması lazım fakat şuan için sadece CS2 için test ettim diğerleri hatalı çalışabilir. <br>

# İndirme

Son versiyonu <a href="https://github.com/Talha-Fe/byno-api-price-checker/releases/latest">buradan</a> indirebilirsiniz.

# Kullanım talimatları

Exe dosyasını indirin. (Exe dosyasını klasörde tutmanızı tavsiye ederim çalıştırdıktan sonra config ve env dosyası oluşturacaktır) <br>
Exe dosyasını çalıştırdıktan sonra konsol açılacaktır açılan konsola API girişi yapın.<br>
Fiyat takibi yapmak istediğiniz itemin adını girin. (ÖNEMLİ: item adını tam olarak yazmanız gerekmektedir aşınım durumu dahil olaraktan. İtem adlarını direkt olarak bynodan kopyalayıp yapıştırabilirsiniz.) <br>
Taban fiyatınızı belirleyin. <br>
İtem belirlediğiniz fiyatın altına düştüğünde konsolda çıkacaktır. <br>
İtem kontrol etme aralığı için bir süre belirleyin.(30saniye ile 60 saniye arası yazmanızı öneririm item sayısı arttıkça saniye artmalı)<br>

# Subject to change

Uygulama şuan optimize değildir API hatası almamak için optimize edilebilir. <br>
